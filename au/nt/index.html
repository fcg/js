2019-2020财年NT北领地州担保

飞出国：北领地Northern Territory是澳大利亚境内一个直属澳大利亚联邦政府的领地，是澳大利亚的两个内陆领地之一，位于澳洲的中北部，首府是达尔文。

北领地是以自然景观和文化特色著称的主要旅游胜地之一，经济以畜牧业、矿业、旅游业和政府服务业为主。

NT北领地州担保申请条件 - 飞出国

满足澳洲DHA基本申请条件

  • 年龄在45岁以下；
  • 通过澳洲技术移民职业评估；
  • 英语至少达到雅思4个6的Competent English水平，除此之外评估/注册机构或NT职业列表可能会有额外的要求；
  • 满足EOI评分65分的评分标准（包括190州担保5分或491偏远地区担保10分）；

满足NT北领地州担保的申请条件

  • 澳洲境内申请人：
    • 只有居住在NT的申请人才能获得NT州担保；
    • 通常申请人在NT至少居住2年才可申请190签证；
  • 澳洲境外申请人：
    • 境外申请人需证明其对NT坚定的承诺以及在NT有良好的就业前景；
    • 通常境外申请人只能申请491签证；
  • 需对NT政府有以下承诺：
    • 自签证授予之日起至少在NT生活和工作三年，从事其技术职业或紧密相关技术职业；
    • 在获得签证或登陆之时让NT政府知晓申请人的签证申请情况；
    • 定期向NT政府更新申请人的联系资料；
    • 参与NT政府进行的调查；
    • NT政府会参考以下因素来评估申请人的州担保申请：
      • 在NT居住的时长；
      • 曾经至NT旅行；
      • 与NT的联系(如有在NT长期居住的亲属）；
  • 职业：
    • 申请人的职业必须在NT北领地移民职业列表之上；或
    • 申请人的职业在当前澳洲政府的合格技术职业清单之上，但不在NT北领地移民职业列表之上，则申请人需提供足够的证据证明其技能职业或紧密相关技能职业在NT具有良好的就业前景；
    • 澳洲境外申请人即使职业在NT移民职业列表之上，也许提供足够的证据证明其职业在NT具有良好的就业前景；
  • 申请人可通过以下途径证明其技能职业或紧密相关技能职业在NT有良好的就业前景：
    • NT雇主提供的工作offer信；
    • 申请人工作技能及经验可满足NT雇主需要的说明；
    • 申请人的职业曾多次出现在NT的职业广告当中，申请人的资历及经验如何与职位空缺相匹配的说明；
    • 来自NT潜在雇主的反馈证据；
  • NT境外申请人需提供相应的资金证明：
    • 必须提供一份以澳币为单位的金融资产清单，并满足以下表格资金要求。

NT境内申请人根据类别不同需满足特定条件

  1. NT毕业生：
    • 申请人在NT的学院或大学完成至少2年的全日制学习并获得学历；
    • 申请人的职业在NT具备良好的就业前景；
    • 申请人需提供NT承诺函，以及六个月的银行流水以证明其具备在NT生活的能力；
    • 表明在北领地具有持续良好的就业前景，在获得签证后有在北领地居住或工作至少三年的真实意图。
  2. 其他国际留学生：
    • 申请人在NT的学习经历未达到2年；
    • 在北领地至少居住六个月；
    • 申请NT州担保前全职从事提名职业或紧密相关职业六个月；
    • 工作期间所得薪水符合NT当地同行业标准；
    • 表明在北领地具有持续良好的就业前景，在获得签证后有在北领地居住或工作至少三年的真实意图。
  3. 打工度假417和462签证持有人：
    • 持有417或462签证至少六个月；
    • 在NT居住并从事提名职业或与提名职业密切相关的职业至少六个月；
    • 工作期间所得薪水符合NT当地的行业标准；
    • 表明在NT具有持续良好的就业前景；
    • 在获得签证后有在北领地居住或工作至少三年的真实意图。
  4. 临时工作457签证及临时技能短缺工作482签证持有人：
    • 申请提名前已在北领地至少居住了六个月；
    • 申请前为担保457签证或482签证的NT企业至少工作六个月；
    • 为申请人担保457签证或482签证的企业需出具声明信，表明：
      • 支持申请人的签证申请；
      • 理解申请人在获得491或190签证后将不再依附于该企业；
    • 表明在北领地具有持续良好的就业前景，在获得签证后有在北领地居住或工作至少三年的真实意图。

NT北领地州担保申请流程 - 飞出国

1· 注册EOI，EOI里选择北领地及对应的签证类别； 2. 通过NT政府在线申请系统递交申请及支持文件； 3. 一旦通过州担保，申请人将收到签证申请邀请函，需在60天之内递交签证申请； 4. 通过Immi Account向DHA递交签证申请； 5. 经过DHA审核，申请人的签证获批。

NT北领地州担保审理周期 - 飞出国

  • 澳洲境外申请大约需要6个月；
  • NT学生或NT境内申请大约需要1个月。

NT北领地州担保申请费用 - 飞出国

300澳币（自2019年12月9日起）

NT北领地州担保材料清单 - 飞出国

所有申请人必须提供以下材料：

  • 相关澳洲职业评估机构出具的提名职业的职业评估函清晰彩色复印件；
  • 详细的个人履历；
  • 学历、技术证书及成绩单的清晰彩色复印件；
  • 主申及起随行家庭成员护照个人信息页的清晰彩色复印件；
  • 如申请人在澳洲境内或曾有澳洲学习或工作经历，需提供签证证明；
  • 如有需要，提供英语能力证明 - NT政府接受DHA认可的所有英语考试，如申请人参加了PTA Academic考试，需在申请中提供分数报告代码；
  • 如需获得配偶加分，需提供配偶职业评估的清晰彩色复印件；
  • 如需获得配偶英语能力加分，需证明配偶英语能力；
  • 对NT的书面承诺函，承诺函需包括以下内容：
    • 自签证授予之时，承诺在NT从事提名职业或紧密相关职业，生活和工作至少三年；
    • 想要在北领地生活的原因；
    • 与澳洲其他城市相比，NT的生活方式更适合申请人；
    • 承诺函不得超过两页。

NT境内申请人还需提供以下材料：

  • 递交申请前六个月的银行对账单，显示在NT的日常开支，证明申请人一直生活在NT；以及
  • NT雇主提供的六个月工资单 -- NT毕业生无需提供。

澳洲境外申请人还需提供以下材料：

  • 如NT职业列表有要求，需提供提名职业或紧密相关职业的NT全职工作offer；
  • 证明申请人已对NT的生活及工作进行过调研，已了解当地的气候、搬家费用和生活费用（将网站复制粘贴到递交的文件中将不被认可）；
  • 表明申请人可在NT轻松找到工作的证据，包括：
    • 申请人所从事的职业或行业在NT劳动力市场的情况描述，以及申请人的技能和经验如何满足NT雇主的需求（需使用个性化的语言描述）；
    • 证明申请人的职业曾多次出现在NT招聘广告当中，说明申请人的资历和经验如何符合实际工作实例的选择标准（不可只使用一两句话简单描述）；
    • 来自NT雇主的关于申请人技能和经验的真实反馈（不可是来自招聘网站及雇主人力资源部门的自动回复）；
  • NT雇主提供的工作offer和/或已签署的劳动合同（工作offer将会被调查与核实 -- 伪造的文件将导致申请被视为欺诈性的）；

  • 证明申请人具备在澳洲定居与生活的财政能力。
