2023-2024财年QLD昆士兰州担保

飞出国：昆士兰州Queensland是澳大利亚的六个州之一，其名字是为了纪念英国的维多利亚女王。昆士兰州位于澳大利亚大陆的东北部，东濒太平洋，西与澳北区及南澳相接，南邻新南威尔士。

昆士兰州面积为172.72万平方公里，约占澳大利亚大陆面积的22.4%，是澳大利亚第二大州。昆士兰州约有54%的面积位于南回归线以北，降雨量少，气候温暖、阳光明媚，有“阳光之州”的美誉。首府布里斯班，是澳大利亚的第三大城市。

昆州2023-24财年州担保于2023年9月5日正式开启，本财年总配额为1550个，其中190担保900个，491担保650个。

境外申请人QLD州担保要求

飞出国：申请昆州州担保，申请人需要满足基本的移民签证要求以及昆州州担保要求：

  1. 移民签证要求：
    • EOI打分达到65分（包括州担保分数）；
    • 持有提名职业的职业评估函；
    • 45周岁以下；
    • 至少雅思4个6；
  2. 昆州州担保要求：
    • 申请昆州190担保，申请人EOI打分达到75分或以上（技工类职业EOI打分达到70分及以上即可）；申请491担保，EOI分数达到65分或以上；
    • 职业在昆州境外担保职业清单QSOL (Offshore Queensland Skilled Occupation List)之上；
    • 语言等同于雅思4个6的水平；
    • 满足职业清单上的工作年限要求。

境内申请人QLD州担保要求

飞出国：该类别要求申请人满足基本的移民签证要求以及昆州州担保要求：

  1. 移民签证要求：
    • EOI打分达到65分（包括州担保分数）；
    • 持有提名职业的职业评估函；
    • 45周岁以下；
    • 至少雅思4个6；
  2. 昆州州担保要求：
    • 申请昆州190担保，申请人EOI打分达到75分或以上（技工类职业EOI打分达到70分及以上即可）；申请491担保，EOI分数达到65分或以上；
    • 职业在移民局移民职业总表上即可；
    • 语言等同于雅思4个6的水平；
    • 如果递交的是190 EOI，则需要证明在递交EOI之前的3个月是居住在昆州且在昆州从事提名职业的学历后的全职工作（不低于30小时每周）；
    • 如果递交的是491 EOI，则需要证明在递交EOI之时是居住在昆州偏远地区且在昆州偏远地区从事提名职业的学历后的全职工作（不低于30小时每周）；
    • 提供将为在昆州注册的昆州雇主继续工作12个月（每周至少30小时）的证据；
    • 针对491担保，则需要该雇佣是在昆州偏远地区；
    • 190承诺在昆州生活和工作2年，491则是3年；
  3. 对于以下职业，有额外要求：
    • 会计（普通）：只有就职于会计公司的会有机会收到邀请，必须是CPA或者CA，或者是在读可以成为CPA或CA；
    • 内审和外审：必须是有内审的工作（不是会计）；
    • 厨师和主厨：不是快餐或者外卖餐馆。

注：

以上提及的昆州雇主需要是在昆州有实体办公室，虚拟办公室不算。

毕业生类别QLD州担保要求

  1. 整体要求：
    • 申请昆州190担保，申请人EOI打分达到75分或以上，申请491担保，EOI分数达到65分或以上；
    • 语言等同于雅思4个6的水平；
    • 当前居住在昆州，如果是申请190，则提名职业要是可以申请190的移民职业；如果是申请491，则需要居住在昆州偏远地区，且职业是可以申请491的移民职业；
    • 学历是100%在昆州完成的，近两年内毕业的；
    • 满足不同学历层次对应的额外要求；
  2. 不同学历层次额外要求：
    • 博士毕业生（190和491担保皆可）
      • 通过了毕业论文答辩；
      • 不需要工作offer；
    • STEM或者农业专业的硕士毕业生（190和491担保皆可）
      • 在昆州全职工作（每周最少30小时），从事的是提名职业或者ANZSCO Skill Level 1的职业；
      • 是以下附表所列的STEM或者农业专业；
    • 非STEM或者农业专业的硕士毕业生（190和491担保皆可）
      • 申请190担保，要求GPA6.0及以上；
      • 申请491担保，要求GPA5.0及以上；
      • 在昆州全职工作（每周最少30小时），从事的是提名职业或者ANZSCO Skill Level 1的职业；
    • 本科毕业生（190和491担保皆可）
      • 申请190担保，要求GPA6.0及以上；
      • 申请491担保，要求GPA5.0及以上；
      • 在昆州全职工作（每周最少30小时），从事的是提名职业或者ANZSCO Skill Level 1的职业。
  3. 对于以下职业，有额外要求：
    • 会计（普通）：只有就职于会计公司的会有机会收到邀请，必须是CPA或者CA，或者是在读可以成为CPA或CA；
    • 内审和外审：必须是有内审的工作（不是会计）；
    • 厨师和主厨：不是快餐或者外卖餐馆。

小生意类别QLD州担保要求

飞出国：昆州小生意类别主要包括以下两个子类别：

  • Pathway 1：申请人在昆州偏远地区收购一个现有10万澳币的小生意，且至少雇佣一位澳洲居民（澳洲公民、澳洲永居或者持有444签证的新西兰公民），在递交EOI之前该小生意已经完成收购且经营至少6个月；
  • Pathway 2：申请人在昆州偏远地区创立一个小生意，且该小生意已运营至少两年，在递交申请之前该小生意是营利的，且近12个月营业额达到20万澳币。

该类别要求申请人满足基本的移民签证要求以及昆州州担保要求：

  1. 移民签证要求：
    • EOI打分达到65分（包括州担保分数）；
    • 持有提名职业的职业评估函；
    • 45周岁以下；
    • 至少雅思4个6；
  2. 昆州州担保要求：
    • 职业在移民局移民职业总表上即可；
    • 申请人居住在昆州偏远地区；
    • 申请人给该小生意全职工作，每周不低于30小时；
    • 申请人在昆州偏远地区持有并运营该小生意；
  3. 对于Pathway 1的额外要求：
    • 收购的小生意在收购前已经在当前的营业场所运营至少两年；
    • 所收购的小生意至少价值10万澳币；
    • 完全支付好10万澳币收购该小生意之后，要证明运营该小生意至少6个月；
    • 至少雇佣一位澳洲居民（每周工作20小时），或者雇佣两位每周工作10小时的员工，员工不能是家庭成员或者外包；
  4. 对于Pathway 2的额外要求：
    • 该小生意已运营至少两年，在递交申请之前该小生意是营利的，且近12个月营业额达到20万澳币；
  5. 对于491小生意类别的特别说明：
    • 不能是股份转让，必须是有企业出售的合同；
    • 不是卖方融资；
    • 不能是向家庭成员或者亲属收购的小生意；
    • 在获签之前不能搬迁或者售卖该小生意；
    • 家庭办公或者虚拟办公不符合该类别要求；
    • 代理商或者特许经营租赁的不符合该类别要求，比如加油站；
    • 在获签后，申请人必须继续在偏远地区经营该小生意至少三年，如果在这之前售卖该小生意，昆州有权撤销申请人的491签证。

QLD昆州190/491申请流程

  1. 境内及境外申请人类别：
    • 申请人确认满足以上要求后，递交EOI，选择昆州；
    • 昆州会审核EOI，通过Email发放邀请；
    • 申请人收到昆州邀请后，需要在14天内通过在线申请昆州州担保；
    • 昆州审核申请人所递交的申请，通过州担保，申请人会收到EOI的签证邀请；
  2. 昆州毕业生类别：
    • 申请人确认满足以上要求后，递交EOI，选择昆州；
    • 申请人递交EOI之后，需要通过昆州官网提交online form以便昆州识别申请人为昆州毕业生申请人；
    • 昆州会审核EOI，通过Email发放邀请；
    • 申请人收到昆州邀请后，需要在14天内通过在线申请昆州州担保；
    • 昆州审核申请人所递交的申请，通过州担保，申请人会收到EOI的签证邀请；
  3. 昆州小生意类别：
    • 申请人确认满足以上要求后，递交EOI，选择昆州；
    • 申请人递交EOI之后，需要通过昆州官网提交491小生意类别评估表；
    • 昆州会审核EOI，通过Email发放邀请；
    • 申请人收到昆州邀请后，需要在14天内通过在线申请昆州州担保；
    • 昆州审核申请人所递交的申请，通过州担保，申请人会收到EOI的签证邀请。
