ACS计算机职业评估 - 飞出国

ACS机构概述 - 飞出国

ACS(The Australian Computer Society)是澳大利亚移民部指定的 ICT职业的移民职业评估机构，对计算机和通信相关职业进行职业评估，相关IT职业只有通过ACS评估才能进行下一步的签证申请或州担保申请。ACS职业评估不需要雅思，因此可以在职业评估的过程中并行准备雅思，这样最节省时间。

相关缩略语解释 - FLYabroad：

  • ICT：Information Communication Technology
  • RPL：Recognition Of Prior Learning
  • PASA：Pre Application Skills Assessment
  • AQF：Australian Qualifications Framework

ACS评估分类 - 飞出国

临时毕业生485签证申请人职业评估

  • 此类职业评估仅限于申请485签证；
  • 此职业评估仅适用于出现在中长期职业列表（MLTSSL)中的提名职业；
  • 申请人需要完成澳大利亚本科以上学历，主修与提名职业高度相关的信息与通信技术（ICT）专业。

有澳洲学习经历申请人职业评估

  • 此类别仅限于申请人已在澳大利亚教育机构获得本科或以上学位，出于移民目的而进行的职业评估；
  • 申请人需要具备与提名职业高度相关的ICT专业澳洲本科或以上学历，并满足下列条件之一：
    • 完成澳洲学历后具备一年相关工作经验（澳洲境内境外均可，必须是完成学历之后的）；或
    • 完成ACS专业年。

主修ICT专业本科或以上学历申请人职业评估

  • 申请人主修ICT专业，与提名职业高度相关，需要在过去10年内具备2年相关工作经验，或者在过去的工作经历中具备任意4年相关工作经验；
  • 申请人主修ICT专业，与提名职业没有高度相关，则需要在过去的工作经历中具备任意4年相关工作经验。

辅修ICT专业本科或以上学历申请人职业评估

  • 申请人辅修ICT专业，与提名职业高度相关，需要在过去的10年中具备5年相关工作经验，或者在过去的工作经历中具备任意6年相关工作经验；
  • 申请人辅修ICT专业，与提名职业没有高度相关，则需要在过去的工作经历中具备任意6年相关工作经验。

主修ICT专业无学位学历以及持有行业认证申请人职业评估

  • 申请人主修ICT专业，与提名职业高度相关，需要在过去的10年中具备5年相关工作经验，或在过去的工作经历中具备任意6年相关工作经验；
  • 申请人主修ICT专业，与提名职业没有高度相关，则需要在过去的工作经历中具备任意6年相关工作经验。

非ICT专业无学位学历及以上学历申请人职业评估

  • 申请人的学历为无学位学历或以上，但非ICT专业，则需要在过去的工作经历中具备任意6年相关工作经验，外加一份合格的RPL申请。

仅具备工作经验的RPL申请人职业评估

  • 申请人不具备高等教育学历，则需要在过去的工作经历中具备8年相关工作经验，外加一份合格的RPL申请。

ACS扣减年限标准 - 飞出国

针对主修ICT专业的澳洲本科及以上学历申请人

  • 申请485签证，无需扣减年限；
  • 申请移民签证，有效工作年限从获得澳洲学历后开始计算，因此无需扣减年限。

针对主修ICT专业的澳洲境外本科及以上学历申请人

  • 专业与提名职业高度相关，近10年内扣减2年，或任意时间段扣减4年；
  • 专业与提名职业没有高度相关，任意时间段扣减4年。

针对辅修ICT专业的澳洲境外本科及以上学历申请人

  • 专业与提名职业高度相关，近10年内扣减5年，或者任意时间段扣减6年；
  • 专业与提名职业没有高度相关，任意时间段扣减6年。

针对主修ICT专业澳洲境外无学位（包括持有行业认证的）申请人

  • 专业与提名职业高度相关，近10年内扣减5年，或者任意时间段扣减6年；
  • 专业与提名职业没有高度相关，任意时间段扣减6年。

针对澳洲境外非ICT专业无学位学历及以上学历申请人

  • 任意时间段扣减6年，并需要出具RPL项目报告。

针对仅具备工作经验无正规学历的RPL申请人

  • 任意时间段扣减8年，并需要出具RPL项目报告。

ACS学历界定 - 飞出国

概述

澳大利亚和海外教育学历都可用于普通和RPL类型的职业评估，只有临时毕业生485签证和曾有澳洲学历的申请人需要具备澳大利亚学历。

学历被评定为主修ICT专业或辅修计算机专业，ICT内容少于辅修内容将被评定为不具备ICT学历。

学历中的ICT课程内容必须达到信息与通信技术专业水平，课程的主要目标是将学生培养成为信息与通信技术专业人员。

ICT主修标准：

  1. 非学位学历要求必须至少有50%的ICT课程内容，少于50%的内容将被认定为不具备ICT学历。
  2. 学士学位要求ICT内容必须贯穿所有学年，并在最后一年达到高级水平；并达到以下标准：
    • 3年制课程具备33%的ICT内容；
    • 4年制课程具备25%的ICT内容；
    • 5年制课程具备20%的ICT内容。
  3. 研究生学历（研究生文凭、硕士或博士学历）要求至少具备ICT专业本科学历才能进入该课程，课程必须具备33%的ICT内容，并且所有内容必须达到研究生水平。

ICT辅修标准

如果ICT内容只能达到主修要求的三分之二，则会被评定为是辅修ICT专业的学历。

非ICT学历

如果ICT内容少于辅修要求，则被评定为不具备ICT学历。

ANZSCO ICT内容

在所有学历中，65%的ICT内容必须是与提名职业高度相关的。

ACS其他服务 - 飞出国

  1. 复审
    • 如果对评估结果不满意，可以在收到职业评估函后60天内申请复审；
    • 申请复审的理由可以是：
      • 想要改变评估的ANZSCO职业代码；
      • 想要提交额外的材料；
      • 对ACS评估结果产生质疑。
    • 复审需要缴纳复审费，需要提交更多的文件以支持申请人的复审申请，如果评估函已用于移民申请则不能进行复审。
  2. 申诉
    • 如果申请人不同意评估结果并且认为评估过程中出现了错误，可以在收到评估结果后60天内提出申诉；
    • 申诉不能再提交额外的文件，申诉成功后相关的费用将被退还；
    • 申诉机会只有一次，并且申诉结果将是最终的。
  3. 加急
    • 自申请之日起，签证截止日期不足12周，可申请加急服务；
    • 必须提前两周告知截止日期；
    • 必须提交证明签证截止日期的PDF文件以获得批准，包括：
      • 带有截止日期的护照印章；
      • 显示签证有效期的签证权利验证在线网页；
      • DIBP通知发放签证的电子邮件。

ACS申请流程 - 飞出国

  1. ACS评估申请材料准备好后，通过ACS在线系统进行在线申请提交，并在线用信用卡支付申请费；
  2. 提交后会收到ACS自动发送的在线申请递交确认信，并可以通过在线系统查询申请状态；
  3. 如果审理过程中需要补交材料，提交后一周内会收到电子邮件的补料信，并在邮件中提供补料的链接，补交的材料提交后继续进入审理流程；
  4. 如果申请过程中没有补料环节的话，一般是递交后一周左右会收到电子邮件发来的ACS评估结果通知信，附件附上申请人的ACS评估函。

详细信息请参考飞出国论坛ACS评估流程。

ACS材料清单 - 飞出国

个人信息文件：

  • 出生证明或护照，仅提供护照个人信息页；
  • 结婚证公证和曾用名公证；（针对婚后更改姓名的申请人）
  • 加急证明。

学历文件：

  • 成绩单公证；
  • 毕业证书或者澳洲学历完成信公证；
  • 如果申请人提供的是微软或思科认证证书，则需要提供认证登录信息以备核查。登录信息可录入空白文档中，转换成PDF格式上传。
  • 基础附加材料，如硕士学历也必须提供本科学历材料。

注：确保学历文件中包含以下信息：

  • 学位或授予证书名称；
  • 大学或授予机构名称；
  • 学位完成或授予证书的日期；
  • 单元或科目名称以及达到的分数；
  • 通过研究完成的学历提供研究摘要。

工作经验文件：

  • 公司信头纸上的推荐信公证件，或第三方法定声明；
  • RPL项目报告。

注：

  1. 推荐信必须包括以下内容：
    • 工作开始和结束的日期 - 如果申请人的工作是“当前的”，工作推荐信应该注明“至今”一词，并且必须包括写推荐信的日期；
    • 工作职责描述 - 需要确定工作经验与提名职业的相关性； * 工作小时 - 全职或兼职；
    • 完成工作的国家 - 如果申请人曾为同一家公司在多个国家工作，则推荐信中必须清楚的显示完成工作的具体日期和相应地点；
    • 公司信头纸以及主管签名。
  2. 法定声明必须包括以下内容：
    • 所有法定声明或宣誓书必须有法定证人清楚签名。文件必须由推荐人说明是在被授权的证人面前宣誓、签署或见证，并且要由被授权的证人签名，注明声明的日期和地点；
    • 法定声明或宣誓书必须由第三方同事撰写，而不能由申请人本人撰写；
    • 由第三方同事撰写的法定声明或宣誓书需要描述其与申请人的工作关系，以及相应工作日期中申请人的工作职责。该同事最好处于更高级别的岗位；
    • 所有第三方法定声明或宣誓书必须包括以下材料之一：
      • 工资单公证 - 最好是第一份和最后一份工资单；
      • 人力资源报表或服务证书公证；
      • 有相应日期的终止信公证。

注：ACS不接受任用书、自我法定声明或合同作为工作推荐信。
