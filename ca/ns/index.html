NSNP项目

新斯科舍省移民提名项目 Nova Scotia Nominee Program ，简称 NSNP ，也就是我们常说的NS省省提名。

NSNP项目分类

NSNP项目重点分类

EE项目 - 经验类（有雇主offer）

申请条件：

  • 在新省至少一年工作经验
  • 年龄21-55岁
  • 拥有加拿大高中或同等学历及以上
  • NOC中0，A类职位语言水平CLB7；NOC中B类职位语言水平CLB5
  • 在IRCC系统中注册EE

详细的介绍请参考飞出国论坛新省省提名-NSNP

EE项目 - 紧缺职业类（有雇主offer）（暂时停止接受申请，飞出国将密切关注开放时间）

NSNP EE项目 - 紧缺职业类（有雇主offer）（NSDEE）在北京时间2017年6月17日凌晨向5月8日抢NSDEE名额时部分保存信息的申请人发出邀请，数量为200多个，邀请其重新递交申请材料。在数周以前部分5月8日顺利提交申请的申请人获得了NS提名。由此可得基本这批次250个 NSDEE的名额已经没有了。未收到邀请的人可以等待下次开放。当前NS EE项目处于关闭状态。飞出国将密切关注开放时间

申请条件：

  • 在IRCC系统中注册EE
  • NSNP评分大于67
  • 拥有新省雇主全职永久技术工作offer
  • 一年相关工作熟练工作经验/在目标职业之一拥有至少一年熟练工作经验
  • 拥有加拿大高中或同等学历及以上
  • CLB7英语或法语水平
  • 有足够的资金在新省定居

详细的介绍请参考飞出国论坛新省省提名-NSNP

常规项目 - 持有雇主offer技术工人类别

申请条件：

  • 拥有新省雇主全职工作offer
  • 有一年相关工作经验（半熟练工和低技能工必须已经拥有六个月雇主经验）
  • 年龄21-55岁
  • 拥有加拿大高中或同等学历及以上
  • 熟练工提供CLB5语言等级证明，母语为英语和法语的除外；半熟练工和低技能工人必须有符合CLB4的语言测试结果
  • 有足够资金在新省定居

详细的介绍请参考飞出国论坛新省省提名-NSNP

常规项目 - 企业家类别

申请条件：

  • 年龄在21岁以上
  • 拥有并积极管理新省业务
  • 净值为60万加币
  • 能够从自己的资金中拿出15万加币在新省投资建立业务
  • 拥有至少三年管理经验且至少三分之一的股份或超过五年的高级业务管理经验
  • CLB听力阅读写作口语成绩至少为五分
  • 表达在新省永久居住的兴趣和意愿并在新省移民局办理申请

详细的介绍请参考飞出国论坛新省省提名-NSNP

常规项目 - 国际毕业生创业类

申请条件：

  • 在新省拥有并活跃的经营自己的企业同时打算长期定居新省
  • 至少有1年连续的管理和拥有当前企业的经验（完全持股）
  • 在新省大学或社区学院完成了至少2学年的全日制课堂学习，同时获得了学位或者文凭
  • 持有有效的毕业后工作许可
  • 语言最低CLB7，即雅思4个6

详细的介绍请参考飞出国论坛新省省提名-NSNP

新省省提名EE紧缺职业类别申请流程 - 飞出国

  1. 在线注册提交或者纸质递交材料；
  2. 收到档案号NS FN；
  3. 获得提名函；
  4. 递交签证申请。

详细的介绍请参考飞出国论坛新省省提名EE紧缺职业类别申请条件及流程。

新省省提名EE紧缺职业类别审理周期 - 飞出国

  • 在线申请：3个月 or more
  • 纸质递交：9个月 or more

新省省提名EE紧缺职业类别申请费用 - 飞出国

NSNP省提名阶段无需申请费。
