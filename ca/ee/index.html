加拿大联邦技术移民EE(Express Entry)

FLY注： 2016年加拿大经济类移民总配额为160,600，其中联邦技术移民EE的配额为58,400，魁北克技术移民配额为26,200，省提名配额为47,800。更详细的移民配额数据请参考飞出国论坛2016年加拿大移民配额计划。

加拿大EE项目介绍

2015年1月1日加拿大引入EE(Express Entry)系统，EE系统是一个在线注册的系统，是一个移民意愿表达系统。申请人首先要注册EE，成功注册EE后会进入候选池，EE下是择优录取，分数高的会被优先选中。被选中的申请人会收到签证邀请，有资格申请加拿大永久居民身份。

新的EE系统与传统的联邦申请的最大区别在于申请人不能主动递交申请，而是要被动的被筛选后，才能进行下一步的签证申请。这样就有效的避免了一些不具备申请资格或者材料不充分的无效申请，大大缩减了移民局审核的时间，使得移民局有更多的时间来处理合格的申请。

EE系统适合于申请联邦技术移民FSW、经验类移民CEC、技工移民FST和部分省的省提名的申请人。

加拿大EE申请类别

  1. 加拿大联邦技术移民 FSW：G类雅思4个6，够67分，最少一年0，A，B类职业工作经验；
  2. 加拿大技工移民 FST：G类雅思听说5，读3.5，写4，1年雇主offer或持有加拿大省政府的职业技工证书，近5年内有2年FST技工经验；
  3. 加拿大经验类移民 CEC：近3年内有1年加拿大0，A，B职业工作经验，B类职业雅思读4其他5，0和A类职业雅思4个6；
  4. 某些省的省提名： 当前开放EE类别（不需要offer）的省有：萨省，新省，新不伦瑞克省，安省。

加拿大EE申请流程

第一步： 进入EE候选池

申请人自我评估如果满足以上列出的前三个类别中的一种（FSW，FST或CEC），即可到移民局官网注册MyCIC账户，填写需要的信息，在线提交EE，进入候选池，同时注册Job Bank，等待移民局的邀请。

第二步： 申请省提名

对于自身分数较低（低于450分）的申请人（EE下的评分情况参考下面的EE评分表格），可以考虑申请EE下的省提名，获得600分的加分，从而可以快速获得移民局的邀请。当前开放过或者正在开放EE下省提名项目的省有：萨省SINP，新省NSNP，安省OINP和新不伦瑞克省NBNP。

第三步： 移民局发邀请ITA

加拿大移民局每个月发两次EE邀请，达到当轮最低邀请分数的申请人，会获得移民局签证邀请ITA。

第四步： 递交签证申请

申请人获得ITA后，需要在60天内递交签证申请。

加拿大EE申请周期

邀请阶段： EE每个月从候选池中筛选2次，达到当次最低邀请分数的人都可以收到邀请。 签证阶段： 收到签证邀请后有60天的期限提交材料。递交EE签证申请后，2个月左右会收到取签信。

加拿大技术移民FSW

如果申请人近10年内从事1年连续的工作，职业属于NOC中Level 0、A或B，且为同一代码，G类雅思达到4个6，有WES学历认证，评分标准达到67分，申请人就可以申请加拿大联邦技术移民(Federal Skilled Workers)。

加拿大联邦技术移民申请条件

一、工作经验：

  1. 至少有一年（1,560 hours total / 30 hours per week）的 工作经验，可以是连续的全职或者等同的兼职；
  2. 有偿的工作（志愿者、无偿的不算）；
  3. 工作属于同一个NOC职业代码（in the same NOC skill type (0, A or B)）；
  4. 工作是近10年的；
  5. 职业属于2011年版NOC的 level 0、A或B。

二、语言能力：雅思达到CLB7的水平（雅思4个6），2年有效期。

三、学历要求：完成加拿大ECA学历认证。

四、评分标准：满足联邦评分67分

加拿大经验类移民CEC

加拿大经验类移民(Canadian Experience Class) 适用于有加拿大留学经历的国际留学生，以及在加拿大境内持有临时工签的申请人。

加拿大经验类移民申请条件

  1. 年龄要求： 主申年满18周岁；
  2. 工作要求： 近3年内至少有1年在加拿大获得的的全职或等同兼职的工作经验（NOC中0、A、B的工作经验）；
  3. 学历学历： 没有学历要求；
  4. 雅思要求： 对于Level 0 和 A 的工作：雅思达到CLB7的水平（雅思4个6），对于Level B 的工作：雅思达到CLB5的水平（读4.0，其他三项5.0）。

加拿大技工移民FST

加拿大技工移民(Federal Skilled Trades)

加拿大联邦技工类移民申请条件

加拿大联邦技工类移民没有学历要求，无自己评分标准，符合下面条件就可以进入EE系统。

  1. 工作经验： 近5年内有2年技工类工作经验（职业类别见下表）；
  2. 工作Offer： 获得至少一年以上的全职雇主offer或加拿大某省颁发的技工执业证书；
  3. 学历学历： 没有学历要求；
  4. 语言要求： 雅思听5，读3.5，写4，说5（说听CLB5，读写CLB4）。
