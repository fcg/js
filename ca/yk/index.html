2018年YNP育空省提名技术移民

飞出国：育空地区(Yukon Territory)是加拿大三个地区之一，位于加拿大的西北方，毗邻美国的阿拉斯加，是以流经该地区的育空河来命名的。首府怀特霍斯（Whitehorse）的名字也来自于育空河，因河上游有急流，水花溅起高如白马故得此名。育空地区拥有大量的野生动物和连绵不绝的大自然风光。

YNP育空省提名移民项目（Yukon Nominee Program YNP）由育空省政府和IRCC联邦移民局合作，符合育空省提名移民条件的申请人可以获得联邦永居签证。育空省提名是一种雇主驱动类型的项目，都需要有工作offer才可以申请。

详细的介绍请参考飞出国论坛育空省提名官方指南。

育空省提名YNP项目分类 - 飞出国

  1. 技术工人类别 Skilled Worker Program
  2. EE类别 Express Entry
  3. 关键影响力工人类别 Critical Impact Worker Program
  4. 商业类别 Business Nominee Program

育空省提名YNP项目介绍 - 飞出国

YNP育空省提名技术工人类别

该类别适用于职业类别属于NOC 0、A、B的申请人。

申请要求 - 飞出国：

  • 申请人的教育和工作经验必须满足NOC的职业要求（最低可接受的教育水平是高中毕业）；
  • 申请人必须提供符合职业标准的高中教育以上文凭、证书、学位等证明，如果申请人正在进行某一评估，需要提供评估机构的名称；
  • 申请人具有至少一年提名职业相关工作经验；
  • 申请人能够提供有效的语言成绩，包括IELTS、CELPIP以及TEF，语言成绩的有效期为两年：
    • 职业类别属于NOC 0、A的申请人，语言需满足：CELPIP至少达到CLB7，IELTS至少达到CLB6，法语TEF至少达到CLB4；
    • 职业累呗属于NOC B的申请人，语言需满足：CELPIP至少达到CLB5，IELTS至少达到CLB5，法语TEF至少达到CLB3；
  • 目前在加拿大的申请人必须在整个在申请期间持有有效的工作许可证；
  • 目前在加拿大持学生许可证的申请人需提供一年的资格证书课程学习证明，或两年的文凭课程学习证书，或完成学士学位课程的证明，；
  • 在育空大学就读ESL课程的外国留学生必须完成ESL课程的第八级。

详细的介绍请参考飞出国论坛育空省提名外国工人需要满足的条件。

YNP育空省提名EE类别

申请条件 - 飞出国：

  • 申请人必须满足进EE的条件，属于FSW、FST或CEC申请类别的一种；
  • 完成在线EE申请，获得IRCC EE Profile Number以及Job Seeker Validation Code;
  • 证明有足够的安家资金以支持申请人及其随行家属在加拿大的生活；
  • 持有有效的育空地区雇主的长期全职工作offer;
  • 有在育空地区生活和工作的计划。

详细的介绍请参考飞出国论坛育空省提名EE类别。

YNP育空省提名关键影响力工人类别

该类别适用于职业类别属于NOC C、D的申请人。

申请条件 - 飞出国：

  • 申请人的教育和工作经验必须满足NOC的职业要求（最低可接受的教育水平是高中毕业）；
  • 申请人必须提供符合职业标准的高中教育以上文凭、证书、学位等证明；
  • 申请人具有至少六个月提名职业相关工作经验；
  • 申请人能够提供有效的语言成绩，包括IELTS、CELPIP以及TEF，语言成绩的有效期为两年：
    • 职业类别属于NOC C、D的申请人，语言需满足：CELPIP至少达到CLB4，IELTS至少达到CLB4，法语TEF至少达到CLB3；
  • 目前在加拿大的申请人必须在整个在申请期间持有有效的工作许可证；
  • 目前在加拿大持学生许可证的申请人需提供一年的资格证书课程学习证明，或两年的文凭课程学习证书，或完成学士学位课程的证明，；
  • 在育空大学就读ESL课程的外国留学生必须完成ESL课程的第八级。

YNP对雇主的要求 - 飞出国：

  • 雇主必须是加拿大公民或者永居居民；
  • 雇主必须在育空地区注册并经营企业至少一年；
  • 雇主必须符合该地区的劳动力标准的最低要求；
  • 雇主必须提供营业执照副本，并且在育空地区企业注册处登记；
  • 获得LMIA豁免的雇主需要提交工作offer信息以及230加币的CIC雇主门户使用费；
  • 雇主必须为申请者提供与育空居民同样的免费健康保险直到申请人有资格获得健康卡为止；
  • 雇主必须按加拿大就业和社会发展规定为申请人提供育空地区中位数的工资。

YNP育空省提名商业类别

申请条件 - 飞出国：

  • 获得至少65分的评分；
  • 完成至少高中以上的教育水平；
  • 具备至少三年的企业主或企业管理经验；
  • 具备五年相关工作经验；
  • 具备至少50万加币的净资产，30万加币的的流动资金，能够证明是合法所得，并且必须由育空的会计事务所核实；
  • 没有被加拿大政府拒签过移民申请，没有申请加拿大其他省或地区的提名；
  • 打算在育空地区长期定居，并且积极的管理和投资企业，企业中必须包含职业类别为NOC 0或A的职业；
  • 证明在最初的两年中如何在企业中投入最低30万加币的资本投资；
  • 申请人对企业拥有完全所有权，或者与加拿大公民或永居居民建立三分之一的合伙关系；
  • 申请人如果是完全或者部分收购现存企业，要求该企业必须是加拿大公民或永居居民所持有，并且在育空地区至少已经营一年；
  • 企业必须属于下列战略部门之一：信息技术、生产制造、增值服务、林业、旅游业、能源、矿业、文化产业、影视制造业。

详细的介绍请参考飞出国论坛育空省提名雇主需要满足的条件。
